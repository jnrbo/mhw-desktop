Install instrutions

1. Git clone this repsitory
2. In github go to the release section
3. Download mhw-desktop-source-with-dependencies-1.0.0-beta.zip so we can get the dependencies.  This is because our nuget is to old to get the packages we need.  
4. Extract the zip file and copy the dependencies, and packages folder to you mhw-desktop directory.
5. Now download Advanced Installer from http://www.advancedinstaller.com/downloading.html, and install


Debug Note:
on the app.config file the setting myHEALTHwareDomain should be set to https://local.myhealthware.com.  Change thsi for testing on local or staging, just be sure to set it back before checkin
